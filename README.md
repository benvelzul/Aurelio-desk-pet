# 🤖 Aurelio – Desk Pet

A small interactive desk robot built with a Raspberry Pi Pico and an SSD1306 OLED display.

Aurelio is not just a display — it has personality, moods, and behaviors that evolve over time.

## ✨ Features

* 👀 Animated eyes with smooth movement
* 👁️ Natural blinking + double blink system
* 😴 Idle → sleepy → sleeping states
* 😠 Reacts to rapid button presses (gets angry)
* 🎲 Randomized behavior for “alive” feeling
* 💥 Screen shake reactions

---

## 🧠 Personality System

Aurelio behaves like a small digital creature:

* **Idle** → Calm and curious
* **Sleepy** → Eyes droop after inactivity
* **Sleeping** → Fully asleep with minimal movement
* **Angry** → Triggered by rapid interaction

---

## ⚙️ Hardware

* Raspberry Pi Pico
* SSD1306 OLED (I2C)
* Push button

---

## 📂 Project Structure

* `main.py` → main behavior loop
* `drawing` → rendering functions
* `logic` → mood + state system *(future improvement)*
